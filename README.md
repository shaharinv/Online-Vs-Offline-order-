Understanding customer preferences and restaurant trends is essential for making informed decisions in the food industry.
In this project, we analyze the Zomato restaurant dataset using Python to uncover meaningful insights.

Objectives

This analysis focuses on answering key business questions:

Do more restaurants provide online delivery compared to offline services?
Which types of restaurants are most preferred by customers?
What is the preferred price range for couples dining out?


Tools & Technologies
Python
Pandas
NumPy
Matplotlib
Seaborn


Key Analysis Performed
Data cleaning and preprocessing
Handling missing and inconsistent values
Feature extraction (e.g., ratings, cost, votes)
Grouping and aggregation analysis
Data visualization for trend identification


Insights (Example)
Identified the most popular restaurant categories
Analyzed customer engagement using votes and ratings
Explored pricing patterns for dining preferences
